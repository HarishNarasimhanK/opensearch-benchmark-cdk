📁 S3 Results

s3://<bucket>/runs/<run-id>/
├── benchmark-results/
│   ├── parquet/            ← CSV (p50, p90, p99, throughput)
│   ├── lucene/
│   ├── parquetLucene/
│   └── benchmark-comparison.html
├── correctness-results/    ← Pass/fail + raw responses
├── data-integrity/         ← Field null/count diffs
├── profiles/               ← Flamegraphs (SVG) every 3 min
└── storage-metrics/

📋 CloudWatch Logs

/opensearch/builder/user-data      ← Build progress
/opensearch/parquet/user-data      ← Setup
/opensearch/parquet/runtime        ← OS runtime
/opensearch/parquet/vmstat         ← Memory (1s)
/opensearch/parquet/node-stats     ← _nodes/stats (10s)
/opensearch/lucene/*               ← Same as parquet
/opensearch/parquetLucene/*        ← Same
/opensearch/benchmark/run          ← Orchestrator
/opensearch/benchmark/parquet      ← OSB parquet
/opensearch/benchmark/lucene       ← OSB lucene

🔍 AOS Metrics Store

SSH tunnel from laptop → benchmark instance → AOS domain:

# Tunnel:
ssh -i ~/opensearch-benchmark.pem \
  -L 5601:<metrics-host>:443 \
  ec2-user@<benchmark-dns>

# Then open: https://localhost:5601/_dashboards
# Index: benchmark-metrics-*

📥 Fetch Results

./scripts/fetch-results.sh <run-id>

# Destroy:
npx cdk destroy --force
